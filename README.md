# restaurant-application

## Idea
A restaurant web application that allows users to browse the menu, explore offers, and make reservations quickly and easily.

## Description
The goal of this project is to create a modern and user-friendly restaurant website.
The application focuses on clear navigation, an attractive visual design, and quick access to important information.

## How to Run the Project

### Prerequisites
- Node.js (LTS version recommended)
- npm

### Steps
1. Open a terminal in the project folder.
2. Install dependencies:

```bash
npm install
```

3. Compile SCSS to CSS:

```bash
npm run compile:sass
```

4. Open [index.html](index.html) in your browser.

Note: each time you edit [src/styles.scss](src/styles.scss), run this command again:

```bash
npm run compile:sass
```
